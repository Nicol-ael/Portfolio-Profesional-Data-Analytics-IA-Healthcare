# 📖 Descripción

Portfolio profesional enfocado en Data Analytics, Inteligencia Artificial y Healthcare Analytics, diseñado para mostrar mi transición estratégica desde el área clínica hacia el ecosistema tecnológico y AI-driven.

Este proyecto integra diseño moderno, branding profesional y presentación técnica de habilidades, experiencia clínica y proyectos relacionados con análisis de datos, automatización e inteligencia artificial aplicada a salud y negocios.

El objetivo principal es construir una identidad profesional sólida dentro del sector Data & AI, mostrando una combinación única entre experiencia healthcare y capacidades analíticas.
